Propósito del sitio

Este proyecto fue creado con el propósito de dar cumplimiento a la fase 3 del curso Diseño de Sitios Web, para ilustrar el uso correcto de las etiquetas básicas de HTML5 y de sus atributos. Además de abordar los conceptos básicos de las hojas de estilo CSS, y el uso de "Grids".
