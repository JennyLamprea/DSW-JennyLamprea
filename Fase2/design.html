Propósito del sitio

Este proyecto fue creado con el propósito de dar cumplimiento a la fase 2 del curso Diseño de Sitios Web, para ilustrar el uso correcto de las etiquetas básicas de HTML5 y de sus atributos

Además de mostrar una pre-visualización de como será el sitio cuando esté completo

Las etiquetas básicas de HTML son:
  • Encabezados HTML, tales como h1, h2, h3. Ayudan a definir los encabezados del más al menos importante
  • Párrafos: Definidos con la etiqueta <p></p>
  • Enlaces: Definidos con la etiqueta <a>
  • Imágenes: Para mostrar imagenes insertadas en la página, se usa la etiqueta <img>
